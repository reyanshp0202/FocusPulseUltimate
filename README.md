# FocusPulse

FocusPulse is a screen-fatigue break assistant for students, gamers, developers, and anyone who works online for long periods of time.

Live demo: https://focus-pulse-pied.vercel.app

## What It Does

FocusPulse works like a check engine light for your focus. It tracks how long you have been working and changes from green to yellow to red as fatigue builds.

- Green: feeling focused
- Yellow: break soon
- Red: break needed

When a break is needed, FocusPulse recommends a short recovery exercise like breathing, stretching, or the 20-20-20 eye rule.

## Features

- Activity modes for studying, coding, gaming, reading, writing, and normal work
- Optional health check-in with Normal, Okay, Stressed, and Tired
- Custom break timing
- Seconds/minutes toggle for real use or quick demos
- Demo mode for judges
- Guided recovery exercises
- Recovery log
- Research-based information page

## Built With

- HTML
- CSS

## Why I Built It

I built FocusPulse because people often stay on screens too long without noticing eye strain, tiredness, or loss of focus. The goal is to make those warning signs easier to notice and help users take healthier breaks.
